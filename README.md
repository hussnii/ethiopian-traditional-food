# Injera Traditional Food – HTML Project

This is a simple **HTML-only project** that introduces **Injera**, the traditional Ethiopian flatbread. The project includes images, a description, nutrition facts, and a feedback form — all written in basic HTML.

## 🌍 About the Project

**Injera** is an important part of Ethiopian cuisine. In this project, I present:
- What Injera is and how it is made
- Ingredients used in Injera
- Common Ethiopian dishes served with Injera
- Nutrition values of Injera (using a table)
- Images of dishes
- A simple user feedback form

## 💡 Features

- Semantic HTML structure (headings, paragraphs, lists, table, form)
- Image embedding with `alt` descriptions
- Internal CSS for table styling
- Feedback form with input fields and radio buttons
- External links to YouTube and tourism pages

## 🛠 Technologies Used

- HTML5
- Internal CSS styling
- No JavaScript (pure HTML project)
